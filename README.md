# Paper_NACratonLitho_FWANT2025
Codes, scripts, and models for paper on the North American cratonic
lithospheric imaging with FWANT.

This repository includes computer codes and scripts (matlab and python)
and velocity model for the paper on NA Cratonic lithosheric imaging with
FWANT, accepted for publication in AGU Advances. More materials will be added upon publication
of the paper.
